## 哈希表

哈希表是一种**键值对（Key-Value）**数据结构，通过**哈希函数**将键映射到数组中的位置，实现 O(1) 时间复杂度的插入、删除和查找。

**常用语言实现**

```python
# 创建
hash_map = {}
hash_map = {"name": "张三", "age": 25}

# 插入/更新
hash_map["city"] = "北京"

# 查询（不存在返回 None）
name = hash_map.get("name")
age = hash_map.get("age", 18)  # 默认值 18

# 删除
del hash_map["age"]
# 或
hash_map.pop("city", None)

# 遍历
for key, value in hash_map.items():
    print(f"{key}: {value}")

# 判断键是否存在
if "name" in hash_map:
    print("存在")
```

`set()` 是 Python 中的**集合**，底层也是基于**哈希表**实现的。它和字典（dict）很像，但**只有键（Key），没有值（Value）**

## 核心特点

| 特性               | 说明                             |
| :----------------- | :------------------------------- |
| **无序**           | 元素没有顺序                     |
| **唯一性**         | 自动去重，元素不重复             |
| **可变**           | 可以增删元素                     |
| **基于哈希表**     | 查找、插入、删除都是 O(1)        |
| **元素必须可哈希** | 列表、字典等不可变类型不能放进去 |
